# SkyChat

The application is a chat, based on AngularJS and Socket.IO
## General commands
### Install dependencies
Use for the package installer
``` 
npm install
``` 

### Start development
To start, use the command
```
npm start
```

Dev server runs at 
```
http://localhost:3000/
```
